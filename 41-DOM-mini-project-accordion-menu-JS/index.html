Web Development Learning Accordion

HTML (Hypertext Markup Language) is the standard language for creating web pages.

Example:

<html>

This is a basic HTML structure used in every webpage.

Learn more about HTML

CSS (Cascading Style Sheets) is used to style the appearance of content written in HTML.

Example:

body {
    background-color: lightblue;
    color: darkblue;
}

This is a simple CSS rule to style the background and text color of a page.

Learn more about CSS

JavaScript is a programming language used to make web pages interactive.

Example:

document.getElementById("demo").innerHTML = "Hello World";

This JavaScript code changes the content of an HTML element with id="demo".

Learn more about JavaScript

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can manipulate the structure, style, and content of the document.

Example:

const element = document.getElementById("demo");

This JavaScript code gets the element with id="demo" from the DOM.

Learn more about DOM

Responsive design ensures that web pages look good on all devices, from desktop monitors to mobile phones.

Example:

@media screen and (max-width: 600px) {
    body {
        background-color: lightgreen;
    }
}

This CSS code changes the background color of the page on smaller screens (e.g., mobile devices).

Learn more about Responsive Design
