🚀 JavaScript Concepts: Filter, Map & Debugger

Open the browser console (Right-click → Inspect → Console) to see how these methods work!

🔍 filter() Method

The filter() method creates a new array filled with elements that pass a test provided by a function. It’s used to extract elements based on specific conditions. 

// Example:
const adults = users.filter(user => user.age >= 18);

🗺️ map() Method

The map() method creates a new array by transforming each element of the original array using a function. It’s useful for changing structure or values. 

// Example:
const names = users.map(user => user.name);

🛠️ debugger Statement

The debugger keyword stops execution of JavaScript and calls the debugging function. This allows developers to inspect and trace code behavior in browser dev tools. 

// Example:
function check(data) {
  debugger;
  return data.map(...);
}
